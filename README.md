# EaseChat 💬

A full-stack real-time chat application built with **React (frontend)** and **Node.js/Express (backend)**, powered by **Socket.io** for real-time communication.

## 📁 Project Structure

---

## 🚀 Tech Stack

### Frontend:
- React
- Tailwind CSS
- Axios
- Socket.io Client

### Backend:
- Node.js
- Express.js
- Socket.io
- Multer (for file uploads)
- MongoDB (Mongoose)
